# VS_Project
VS_Project - E-commerce Website

ASP.NET Core e-commerce website for selling product with SQL Server database.

## Features
- Product catalog
- User authentication
- Shopping cart
- Order management

## Technology Stack
- ASP.NET Core 8.0
- Entity Framework Core
- SQL Server
- Bootstrap 5

## Setup
1. Clone repository
2. Update database connection string in appsettings.json
3. Run `dotnet run`
